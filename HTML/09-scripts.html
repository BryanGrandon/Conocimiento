La formas de integrar codigo JavaScript

Script en linea
El código JS se incluye en el HTML, dentro de la etiqueta script.
Script externo
El código JS se incluye en el fichero Javascript enlazado en src.

Modo de carga del string

Cuando indicamos un script externo mediante el atributo src, por defecto el proceso de carga del script por parte del navegador es el siguiente:

Detiene temporalmente la carga del HTML en el lugar donde se encontro el script.
Descarga el script .js al cache de ficheros temporales del navegador.
Ejecuta el script una vez descargado.
Reanuda la carga del documento HTML por donde lo dejo, en cuanto termina la ejecucion

Modo de carga

Carga normal
Atributo (ninguno)
Bloquea la carga de la página y se ejecuta inmediatamente.

Carga asincrona
Atributo (async)
Ejecuta el script tan pronto como esté disponible.

Carga diferida
Atributo (defer)
Aplaza la ejecución del script al final de la carga de la página.
