Selectores de CSS

Selectores Basicos

Universal -> * { }
Se aplica a todos los elementos del documento.

De etiqueta -> etiquetas html
Se aplica a los mismos elementos de la etiqueta.

De class -> .class { }
Se aplica a los elementos que tengan la misma class

De id -> #id { }
Se aplica al unico elemento id

Selectores Avanzados

De agrupacion -> a, b { }
Se aplica a los mismos elementos de las etiquetas.

Desendiente -> a b { }
Se aplica al ( b ) si esta dentro del ( a )

Adyacente -> a + b { }
Se aplica si ( b ) sigue directamente al ( a )

General -> a ~ b { }
Se apica al ( b ) si tiene al mismo padre comun que ( a )

Hijo -> a > b { }
Se aplica al ( b ) si son hijos directos de ( a )

:Pseudoclases

:link -> link
Aplica estilos cuando el enlace no ha sido visitado

:visited -> Google
Aplica estilos cuando el enlace ha sido visitado

:hover -> hover
Aplica estilos cuando pasamos el cursor sobre un elemento

:active -> active
Aplica estilos cuando pulsamos sobre el elemento

:Pseudoelemento

content -> content: "";
Propiedad para generar contenido a ::before o ::after

::before -> actual
Aplica los estilos antes del elemento indicado

::after -> actual
Aplica los estilos despues del elemento indicado

::first-letter ->

First letter

Aplica los estilos en la primera letra del texto

::first-line ->

first line

Aplica los estilos en la primera linea del texto

::selection ->

seleccionar

Aplica estilos al fragmento de texto seleccionado por el usuario
