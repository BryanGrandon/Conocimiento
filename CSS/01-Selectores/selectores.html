Selectores de CSS

Selectores Basicos

Universal -> * { }
Se aplica a todos los elementos del documento.

De etiqueta -> etiquetas html
Se aplica a los mismos elementos de la etiqueta.

De class -> .class { }
Se aplica a los elementos que tengan la misma class

De id -> #id { }
Se aplica al unico elemento id

Selectores Avanzados

De agrupacion -> a, b { }
Se aplica a los mismos elementos de las etiquetas.

Desendiente -> a b { }
Se aplica al ( b ) si esta dentro del ( a )

Adyacente -> a + b { }
Se aplica si ( b ) sigue directamente al ( a )

General -> a ~ b { }
Se apica al ( b ) si tiene al mismo padre comun que ( a )

Hijo -> a > b { }
Se aplica al ( b ) si son hijos directos de ( a )
